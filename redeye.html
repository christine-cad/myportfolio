Reflection

Reflection

This project pushed me in term of learning how to use UI prototyping software (Adobe XD) and new coding skills(HTML/CSS/JavaScript) in a short period. Although I believe my redesign is an improvement from the previous site, because the focus was on learning these new skills, time to research for user experience and improve the graphic design was left as a secondary priority. I believe if more time were provided, these key elements would have enhanced, creating a higher quality site. That being said, the project has plenty of aspects I am happy with. At the beginning of this project, I had barely used Adobe XD before; however, at the end of this project, I could confidently say I could operate the program comfortability. This was a significant catalyst in further improving my skills in the program, which I believe is now at a high level. Additionally, learning basic HTML CSS and JavaScript helped me understand more about how frontend development operates and is created. Although there are improvements to be made to this redesign, this project helped me gain new skills and a clearer insight on user interface development.
